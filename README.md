# gymio

Gymio is a modern membership and attendance management app built with Next.js, Supabase and Tailwind CSS. It focuses on giving small gyms a clean, fast interface to track members, subscriptions and daily activity without bloated, oldâ€‘school software

## Getting started

1. Clone the repository  
2. Create your `.env` file with database and auth configuration  
3. Install dependencies  
   - `pnpm install`
4. Run the development servers  
   - `pnpm run dev`
